# Draglo

An innovative and intuitive platform for arranging travel plans with Google Place Search and drag-and-drop calendar.


Website URL: https://draglo.com

### Back-End
* Node.js with Express
* RESTful API
* Reverse Proxy by NGINX
### Front-End
* HTML, CSS, JavaScript
* Bootstrap
* AJAX
* Webpack
### Database
* MySQL
* Redis
### Framework
* MVC
### Algorithm
* K-means Algorithm
### Cloud Service (AWS)
* Elastic Compute Cloud (EC2)
* Relational Database Service (RDS)
### Networking
* HTTPS / SSL
* Domain Name System (DNS)
### Test
* Mocha
* Chai
* Nock
### 3rd Party APIs and Open Source Tools
* Socket.io
* FullCalendar

* Google OAuth Login
* Google Maps API
* Goolge Direction API
